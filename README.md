# MT Game Assistant

> 为卡牌手游《我叫MT Online》开发的可视化挂机脚本系统

## 项目简介

MT Game Assistant 是一个基于可视化编程的挂机脚本系统，允许用户通过图形界面配置挂机逻辑，无需编写代码即可实现自动化游戏操作。

### 核心特性

- **可视化编程** - 基于 React Flow + Tarui 的拖拽式节点编辑器
- **实时调试** - 前端与后端通过网络通信实现双向状态同步
- **逻辑导出** - 支持将可视化逻辑导出为 JSON 配置文件
- **跨平台** - 后端基于 Airtest，支持 Android 模拟器和真机
